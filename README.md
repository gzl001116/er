# 博客项目

这是一个使用 Hexo 构建的博客项目，并已部署到 GitHub Pages。

## 本地开发

1. 安装依赖：
   ```
   npm install
   ```

2. 生成静态文件：
   ```
   npx hexo generate
   ```

3. 启动本地服务器：
   ```
   npx hexo server
   ```

4. 访问 http://localhost:4000 查看博客

## 部署

要将博客部署到 GitHub Pages，请运行：
```
npx hexo deploy
```

## 项目结构

- `source/_posts/` - 博客文章目录
- `themes/` - 主题目录
- `_config.yml` - 全局配置文件
